# NG Challenge 

El siguiente repositorio contempla los 5 puntos solicitados en la consigna del desafio. Sistema básico el cual contempla un lado servidor y otro cliente que trabaja como ecosistema para lograr un manejo optimo de información desde un servicio externo.

The following repository covers the five points requested in the challenge instructions. Basic system that includes a server side and a client side that works as an ecosystem to achieve optimal information management from an external service.

## PASOS PARA EJECUTAR EL PROYECTO / STEPS TO RUN THE PROJECT:
1) Asegurarse de tener instalada la ultima version de Nodejs. Ensure you have the latest version of Nodejs installed. Link oficial: https://nodejs.org/es/download
2) Clonar el repositorio. Clone the repo.
```bash
   git clone https://github.com/VainstubTomas/NG.git
```
3) Crear un archivo .env en base al archivo .env.example. Create an a .env file based from .env.example
4) Dirigirse a la carpeta "frontend" y ejecutar el siguiente comando. Go to the “frontend” folder and run the following command. 
```bash
   npm install
```
5) Dirigirse a la carpeta "backend" y ejecutar el siguiente comando. Go to the “backend” folder and run the following command.
```bash
   npm install
```
6) Instaladas las dependecias. Install dependencies:
6.1) Desde la terminal dirigirse a la carpeta "frontend" y ejecutar. From the terminal, go to the “frontend” folder and run:
```bash
   npm run dev
```
6.2) Desde la terminal dirigirse a la carpeta "backend" y ejecutar. From the terminal, go to the “frontend” folder and run:
```bash
   nodemon src/app.js
```
